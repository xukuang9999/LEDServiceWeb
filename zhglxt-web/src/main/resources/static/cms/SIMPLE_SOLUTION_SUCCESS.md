### **Global Functions Available**
```javascript

```

## 🎯 **How It Works**

1. **Button Click** → Calls `showNewsModal(newsId)`
2. **Content Extraction** → Finds existing Bootstrap modal with same ID
3. **Modal Creation** → Creates simple modal if not exists
4. **Content Display** → Shows extracted title and content
5. **User Interaction** → User can read, close with ESC or click outside
